# 鲁棒符号图神经网络

## 模型运行步骤

1. **数据准备**
   - 准备pickle格式的符号图数据集
   - 数据集应包含图边、边符号、节点特征以及训练/验证/测试划分

2. **模型训练**
   - 运行main.py并指定data_path参数
   - 根据需要指定超参数

3. **关键组件**
   - 第一阶段：SGCNEncoder用于初始节点嵌入
   - 第二阶段：使用Gumbel-Sigmoid进行可学习边剪枝
   - 第三阶段：融合节点嵌入和边预测器

## 基本命令

```bash
python main.py --data_path 数据集路径
```

## 主要超参数

- `--data_path`: 数据集路径（必需）
- `--hidden`: 隐藏层维度（默认：64）
- `--epochs`: 训练轮数（默认：1000）
- `--lr`: 学习率（默认：0.001）
- `--fusion`: 嵌入融合方法（默认：concat）
- `--l1_reg_weight`: L1正则化权重（默认：0.0）
- `--temp_start`: 初始温度（默认：3.0）
- `--temp_end`: 最终温度（默认：0.5）

## 输出

- 训练进度打印到控制台
- 最终结果保存到带时间戳的文本文件
- 指标包括F1、准确率和AUC
